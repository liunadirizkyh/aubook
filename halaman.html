Chapter 1: Python Basics

Abstract

This chapter will get you started with Python, using it as a calculator, managing Python packages, and helping you familiarize yourself with the Jupyter notebook. As you will see, Python has a great set of packages with expansive sets of functionalities. At the end of this chapter, you should be able to execute commands in Python, install and manage the Python packages in Jupyter notebook, and use Python's basic mathematical functions.


1.1 Getting Started With Python

The first step in using Python is to set up the working environment on the computer. This section introduces the initial processes to get it started.


There are different ways to install Python and related packages, and we recommend using Anaconda1 or Miniconda2 to install and manage your packages. Depending on the operating system (OS) you are using (i.e., Windows, Mac OS X, or Linux), you will need to download a specific installer for your machine. Both Anaconda and Miniconda are aimed at providing easy ways to manage the Python work environment in scientific computing and data sciences.
